What is NBASIC?
===============

  NBASIC is an attempt to create a portable BASIC interpreter with behavior
  similar to traditional versions of Microsoft BASIC.

Why write NBASIC?
=================

  I wanted a way to have the same BASIC programming environment on various
  machines that did not entail running an emulator.

Why NBASIC?
===========

  New BASIC, or my last name, or N is the next letter after M; pick one.

What is the license?  Is it open-source?  Is it free?
